## Project Structure

- `controllers`: stores controllers for each api
- `test`: stores unit test
- `public`: stores statics files
- `db`: stores database schema
- `views`: stores templates

## Install Dependencies

```shell
yarn install
```

## Run

### Dev Mode

```shell
yarn start
```

## Testing

```shell
yarn test
```
